## 🚀 Project Overview  

This project is bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) and uses **Next.js + TypeScript + Tailwind CSS**.  

- Optimized for fast page loads and SEO  
- Uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) for automatic font optimization (Geist font)  
- Fully responsive, mobile-first UI
